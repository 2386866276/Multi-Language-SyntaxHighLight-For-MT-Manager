## English

### Important – Please Read First
MT Manager **already includes built-in syntax highlighting for most common languages**.  
- **If you only need basic code highlighting**, the default setup is sufficient – you don’t need to install this plugin.  
- **If you want a specific colour theme** (e.g. eye-friendly dark mode, IDE-inspired colours, or a custom palette), then this plugin provides precisely that – a carefully designed colour scheme for both light and dark modes.

### Overview
A collection of custom syntax highlighting files for **MT Manager**, covering **PHP, HTML, CSS, C, C++ and C#**.  
Each language uses the same unified colour palette for a consistent visual experience across different file types.

### Supported File Extensions
| Syntax File | Extensions |
|------------|------------|
| PHP.mtsx | `.php` |
| HTML.mtsx | `.html`, `.htm` |
| CSS.mtsx | `.css` |
| C.mtsx | `.c` |
| C++.mtsx | `.cpp`, `.cxx`, `.hpp`, `.hxx`, `.cc`, `.h` |
| C#.mtsx | `.cs` |

### Features
- Consistent colour scheme across all supported languages.
- Carefully tuned colours for strings, numbers, comments, keywords, operators, functions, and more.
- Special comment markers (`TODO`, `FIXME`, `WARNING`) highlighted in distinct colours.
- Compatible with both light and dark editor themes.

### Installation
1. Download the desired `.mtsx` file(s) from this repository.
2. In **MT Manager**, navigate to `/MT2/syntax/` (create the folder if it doesn’t exist).
3. Copy the file(s) into that directory.
4. Force‑close MT Manager (swipe it away from recent apps) and reopen it.
5. Open a supported file – the new highlighting will be applied automatically.

### Customisation
You can edit the `styles` section of each `.mtsx` file with any text editor.  
Each style is followed by two hex colour values: the first for light mode, the second for dark mode.  
After editing, restart MT Manager to apply the changes.

## 简体中文

### 重要提示 – 请先阅读
MT 管理器 **原生已支持大多数常见语言的语法高亮**。  
- **如果你只是想在 MT 里查看代码**，默认高亮完全够用，无需安装本插件。  
- **如果你想要一套特定的配色风格**（比如深色护眼、仿 IDE、仿 VSCode 或作者自定义的颜色），这个插件才有安装价值 – 它提供的就是这套精心配色的高亮方案。

### 简介
一组专为 **MT 管理器** 设计的语法高亮文件，覆盖 **PHP、HTML、CSS、C、C++、C#** 六种语言。  
所有语言使用统一的配色方案，确保在不同文件间切换时视觉效果一致。

### 支持的文件扩展名
| 语法文件 | 扩展名 |
|---------|--------|
| PHP.mtsx | `.php` |
| HTML.mtsx | `.html`, `.htm` |
| CSS.mtsx | `.css` |
| C.mtsx | `.c` |
| C++.mtsx | `.cpp`, `.cxx`, `.hpp`, `.hxx`, `.cc`, `.h` |
| C#.mtsx | `.cs` |

### 功能特性
- 所有语言使用统一的配色方案，视觉一致。
- 字符串、数字、注释、关键字、运算符、函数名等均有独立颜色。
- 注释内的特殊标记（`TODO`、`FIXME`、`WARNING`）分别着色，一目了然。
- 兼容浅色与深色模式。

### 安装方法
1. 从本仓库下载所需的 `.mtsx` 文件。
2. 在 MT 管理器中进入 `/MT2/syntax/` 目录（如不存在请手动创建）。
3. 将文件复制到该目录内。
4. **彻底退出** MT 管理器（从最近任务中划掉），再重新打开。
5. 打开对应类型的文件，新语法高亮即会自动生效。

### 自定义配色
使用任意文本编辑器打开 `.mtsx` 文件，修改 `styles` 部分。每个样式名后跟两个十六进制颜色值：第一个用于浅色模式，第二个用于深色模式。  
修改后重新启动 MT 管理器即可生效。

## 繁體中文（港澳臺）

### 重要提示 – 請先閱讀
MT 管理器 **原生已經支援大多數常見語言嘅語法高亮**。  
- **如果你只係喺 MT 入面睇下代碼**，預設嘅高亮已經好夠用，唔使裝呢個插件。  
- **如果你想要一套特定嘅配色風格**（例如深色護眼、仿 IDEA、仿 VSCode 或者作者自訂嘅色系），先至值得裝 – 呢個插件就係提供呢套配色嘅高亮方案。

### 簡介
一組專為 **MT 管理器** 設計嘅語法高亮檔案，涵蓋 **PHP、HTML、CSS、C、C++、C#** 六種語言。  
所有語言採用統一嘅配色方案，確保唔同檔案之間嘅視覺效果一致。

### 支援嘅副檔名
| 語法檔案 | 副檔名 |
|---------|--------|
| PHP.mtsx | `.php` |
| HTML.mtsx | `.html`, `.htm` |
| CSS.mtsx | `.css` |
| C.mtsx | `.c` |
| C++.mtsx | `.cpp`, `.cxx`, `.hpp`, `.hxx`, `.cc`, `.h` |
| C#.mtsx | `.cs` |

### 功能特色
- 所有語言採用統一配色，視覺一致。
- 字串、數字、註解、關鍵字、運算符、函數名等均有獨立顏色。
- 註解內嘅特殊標記（`TODO`、`FIXME`、`WARNING`）分別著色，一目了然。
- 兼容淺色同深色模式。

### 安裝方法
1. 由呢個 repository 下載所需嘅 `.mtsx` 檔案。
2. 喺 MT 管理入面進入 `/MT2/syntax/` 目錄（如果冇嘅話請手動建立）。
3. 將檔案複製到嗰個目錄入面。
4. **徹底結束** MT 管理器（從最近應用程式列表中劃走），然後再重新開啟。
5. 開啟對應類型嘅檔案，新嘅語法高亮就會自動生效。

### 自訂配色
你可以用任何文字編輯器打開 `.mtsx` 檔案，修改 `styles` 部分。每個樣式名後面會跟住兩個十六進制顏色值：第一個係淺色模式用，第二個係深色模式用。  
修改後重新啟動 MT 管理器即可套用。
