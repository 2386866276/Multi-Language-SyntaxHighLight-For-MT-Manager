# Multi-Language-SyntaxHighLight-For-MT-Manager

MT 管理器多语言语法高亮配色方案 – 包含 PHP、HTML、CSS、C、C++、C# 等语言的统一配色高亮文件。

Multi-language syntax highlighting color scheme for MT Manager – includes unified highlight files for PHP, HTML, CSS, C, C++, C#, and more.

## 包含语言 / Supported Languages
- PHP
- HTML
- CSS
- C
- C++
- C#

## 安装 / Installation
将 .mtsx 文件放入 /MT2/syntax/ 目录，强制退出 MT 管理器后重新打开即可。

Place the .mtsx files into /MT2/syntax/, then force-stop and restart MT Manager.
